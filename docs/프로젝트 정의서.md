# 프로젝트 정의서

## 프로젝트 명칭
**sharing is caring**

## 개발 시스템 명칭
**미니 드라이브**

## 개발 시스템 설명
업무 문서 공유를 위한 클라우드 기반 파일 관리 시스템으로 파일 접근 및 최신 버전 관리를 용이하게 한다. 외부 협력 업체와 파일 공유 시 접근 권한 관리나 외부 유출 위험 등의 보안 문제도 해결한다.

## 핵심 기능
- 중앙 관리 가능
- 직관적이고 간단한 업로드 및 다운로드
- 파일 공유
- 파일 검색
- 어느 정도의 버전 관리
- 사용자 계정
- 보안 기능
- 웹 브라우저로 접근
