# 시스템 품질 요소 측정

## 사용자 기대치
- 정확성: 파일 업로드/다운로드, 공유 기능 등이 제대로 작동
- 사용성: 직관적이고 쉬운 사용
- 효율성: 빠른 파일 검색
- 보안성: 로그인 인증, 파일 접근 권한 제어, 암호화된 데이터 전송

## 개발자/유지보수 담당자 기대치
- 유지보수성: 읽기 쉬운 코드, 수정 및 기능 추가가 용이한 구조
- 테스트 가능성: 특정 기능을 테스트하기 쉬운 구조

## 개발 조직 관리자 기대치
- 적용성: 소프트웨어의 일부 또는 전체가 향후 프로젝트에 사용될 수 있기를 원함
- 가시성: 소프트웨어 개발 진행 상황을 쉽게 파악
- 생산성: 효율적인 개발
